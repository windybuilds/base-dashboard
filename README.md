# Base Asset Dashboard 🚀

一个基于 Base 网络的去中心化资产看板，实时展示钱包余额与资产分布。

## ✨ 功能特点
- **实时看板**：通过 ECharts 可视化展示 Base 链上资产分布。
- **钱包交互**：支持 MetaMask 等 Web3 钱包的一键连接。
- **只读安全**：采用非托管模式，仅读取链上公开数据，保障用户私钥安全。
- **响应式设计**：支持 PC 与移动端流畅访问。

## 🛠️ 技术栈
- **前端框架**: Vue 3 (Vite)
- **链上交互**: Ethers.js
- **数据可视化**: ECharts
- **部署平台**: Vercel
